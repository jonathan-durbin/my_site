Background

In the fall semester of my junior year of undergrad, I worked with a friend on an independent study about agent-based modeling. I didn’t have a lot of programming experience at the time, nor did I want to. We used a software called NetLogo (good language for agent-based modeling) in our study. My friend, who had a bit more programming experience than I did, realized that this software would be good for generating the Mandelbrot Set and Julia Sets. Below is one Julia Set he generated using NetLogo:

Naturally, I became very interested in fractals after I saw this; I copied the code he wrote and excitedly started exploring different Julia Sets even though I didn’t have much experience with the code itself. I generated a few more neat looking Julia Sets for myself! More that I made can be found here, I just don’t have space for all of them in this article.

If you would like to see the files that generated these fractals, they can be found here, but you’ll have to install NetLogo to run them.

Getting Into It

I get a stronger interest in learning how to code after completing the independent study. After some research I decided to learn python for its simplicity and large user base. I learned the algorithm for generating these fractals (practically the same algorithm) by reading various articles on the Mandelbrot Set and Julia Sets. I will spend the next section explaining how I understand the algorithm, but these articles may be more helpful to some:

  • Wikipedia has some interesting ideas for generating Julia Sets, but it’s also Wikipedia. Somehow incomplete and overcomplicated at the same time.
  • University of Toulouse I missed this article when I was exploring Julia Sets. Fascinating read! I will definitely go back and read through it again sometime.
  • Kukuruku gets into the theory behind the algorithm, which can be helpful.
